# shuoshuo
树洞说说
### 技术栈：Bootstrap + JavaStript + MongoDB + Express + ejs
- 用户进行登录、注册的过程中，后台数据库进行查询和匹配用户信息，处于对用户个人信息的考虑，使用了MD5加密。
